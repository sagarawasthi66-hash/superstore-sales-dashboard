# Superstore Sales Dashboard and Data Visualization

## Overview

This project focuses on data visualization and storytelling using the Superstore sales dataset. The objective is to analyze sales performance, profit trends, customer segments, and product insights using Python visualizations.

---

## Objectives

* Create meaningful visualizations
* Identify business trends
* Perform sales analysis
* Present insights through storytelling
* Build dashboard-style reports

---

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

---

## Visualizations Created

* Sales by Region
* Profit by Category
* Monthly Sales Trend
* Top 10 Products by Sales
* Customer Segment Distribution

---

## Key Business Insights

* The Western region generated the highest overall sales.
* Technology category contributed maximum profit.
* Monthly trends revealed seasonal sales fluctuations.
* Consumer segment accounted for the majority of customers.
* Top-selling products significantly impacted total revenue.
